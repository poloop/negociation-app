# negotiation-app

## Project setup

```bash
npm install
```

### Compiles and hot-reloads for development

```bash
npm run serve
```

### Compiles and minifies for production

```bash
npm run build
```

### Run your unit tests

The test framework for this application is Mocha.

```bash
npm run test:unit
```

### Run your end-to-end tests

The test framework for this application is Cypress.

```bash
npm run test:e2e
```
